# Muhammad Awais - Personal Brand Website

## Structure
- 7 HTML pages: index, about, services, case-studies, portfolio, testimonials, contact
- 1 shared CSS (styles.css)
- 1 shared JS (main.js)
- Color scheme: White, Dark Navy (#0A0F2C), Gold (#D4AF37)

## Pages & Key Sections
1. index.html – Hero, Trust Metrics, Services Grid, Process, Upwork Section, CTA
2. about.html – Story, Skills, Certifications
3. services.html – Detailed Service Cards
4. case-studies.html – Results with metrics
5. portfolio.html – Project gallery
6. testimonials.html – Reviews + Upwork button
7. contact.html – Form, WhatsApp, Calendly CTA

## Design System
- Fonts: Inter (body), Playfair Display (headings)
- Glassmorphism cards, smooth scroll, sticky nav
- Gold accent gradients, dark navy backgrounds
- Mobile-first responsive
